This repository contains source data and codes for the AroMIP project. AriMIP is a sequence-based method for predicting the 
membrane insertion propensities of aromatic-centered motifs in intrinsically disordered proteins. The method is described in:

Fidha Kunnath Muhammedkutty and Huan-Xiang Zhou (2026), A membrane insertion code for intrinsically disordered proteins, bioRxiv.
